Esqueleto BÁSICO para começar QUALQUER HTML:
  <!DOCTYPE html>              Essa tag informa que o documento é HTML5
  <html lang="pt-br">             Nessa tag iniciamos o documento HTML, já configurado como Português do Brasil
	<head>                         Início da tag HEAD, onde colocamos as configurações do nosso documento HTML e/ou CSS
	   <meta charset="UTF-8">            Codificação binária UTF-8, que pode representar qualquer caractere universal padrão do Unicode
	   <title>Título da página</title>     Aqui configuramos o título da página.
	</head>                       Fechamento da tag HEAD
	<body>                        Início da tag BODY, onde incluímos todo o conteúdo visível do site.
	   <h1>Título</h1>       Apenas um exemplo de título.
	   <h4><p>Texto para visualização!</p></h4>     Apenas um exemplo de conteúdo
	</body>                      Fechamento da tag BODY
  </html>                             Fechamento da tag HTML

   Caso queira apenas copiar e colar esse código num documento HTML,
   apague o que estiver em itálico ou adicione a tag <!-- antes e --> no final do texto que não quer que apareça no navegador.
   O que colocar no meio de <!-- e --> vai se transformar em comentário e vai aparecer apenas no próprio código HTML.


	
-Ajustar página automaticamente:
	<meta name="viewport" content="width=device-width,initial-scale=1.0">

	
-Exemplo de tag para deixar o CSS em um arquivo separado:
	<link rel="stylesheet" type="text/css" href="css/estilos.css">

	
Esqueleto BÁSICO + ajuste automático da página + arquivo CSS em arquivo separado:
  <!DOCTYPE html>
  <html lang="pt-br">
	<head>
	   <meta charset="UTF-8">
	   <title>Título da página</title>
	   <meta name="viewport" content="width=device-width,initial-scale=1.0">
	   <link rel="stylesheet" type="text/css" href="css/estilos.css">
	</head>
	<body>
	   <h1>Título</h1>
	   <h4><p>Texto para visualização!</p></h4>
	</body>
	</html>

	Para funcionar esse exemplo de arquivo CSS citado acima, você deve criar um arquivo "estilos.css"
	dentro de uma pasta com título "CSS" e colocar no mesmo diretório do seu arquivo HTML".

pre {
	border: 1px solid   #cccccc;           border: Define a largura, o estilo e a cor da borda do elemento.
	font-family: Arial, Helvetica, sans-serif;     font-family: Define a fonte usada para o texto dentro do elemento.
	font-weight: bolder;                       font-weight: Define a espessura da fonte usada para o texto dentro do elemento.
	font-size: 17px;                             font-size: Define o tamanho da fonte usada para o texto dentro do elemento.

	padding: 3px;                              padding: Define o preenchimento interno do elemento.

	margin-right: 10px;                    margin-right: Define a margem direita do elemento.
	white-space: pre-wrap;             white-space: Define como o espaço em branco é tratado dentro do elemento.
	}

Conhece a tag de Itálico especial?
	A tag DFN é uma tag HTML que representa o elemento de definição.
	Ela é usada para marcar um termo que será definido dentro do conteúdo.
	O elemento pai mais próximo da tag DFN também deve conter a definição/explicação do termo.
	O termo dentro da tag DFN pode ser qualquer um dos seguintes:

		1. Apenas o conteúdo do elemento DFN.
		2. Com o atributo title adicionado.
		3. Com uma tag ABBR dentro do elemento DFN.
		4. Com o atributo id adicionado.

	**Aqui está um exemplo de como usar a tag DFN:
		Digite no seu código:
<dfn title="HyperText Markup Language">HTML</dfn> é a linguagem de marcação padrão para criar páginas da web.
		...para ficar assim:
	HTML é a linguagem de marcação padrão para criar páginas da web.

	**Ou você pode usar com CSS e ficar assim:

	Tag DFN com CSS

	Foi usado essas propriedades na tag DFN para ficar desse jeito:
	dfn {
		border: 1px solid rgb(0, 0, 0);
		padding: 2px;
		font-size: 43px;
		background-color: rgba(0, 217, 255, 0.541);
		color: rgb(94, 0, 73);
		}
